# Quick Start Guide

## Installation

```bash
cd CivilAviationMockExam-Website
npm install
```

## Run Development Server

```bash
npm run dev
```

Visit: http://localhost:3000

## Build for Production

```bash
npm run build
npm start
```

## What's Included

✅ Professional design with aviation theme
✅ All 8 exam sections displayed
✅ PPL information and explanation
✅ SEO optimized (meta tags, structured data)
✅ Responsive design (mobile, tablet, desktop)

## Customization

1. **Update Images**: Replace Unsplash URLs in components with your own images
2. **Change Colors**: Edit `tailwind.config.js`
3. **Update Content**: Edit files in `components/` directory
4. **SEO**: Update metadata in `app/layout.tsx`
