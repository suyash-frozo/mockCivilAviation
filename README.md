# Civil Aviation Mock Exam Website

Professional website for the Civil Aviation Mock Exam mobile app. Built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- **SEO Optimized**: Meta tags, structured data, and semantic HTML
- **Professional Design**: Modern, aviation-themed design with flight imagery
- **Responsive**: Works perfectly on all devices
- **Fast**: Built with Next.js for optimal performance
- **Accessible**: Follows web accessibility best practices

## Sections Covered

1. **Air Law** - Regulations and legal aspects
2. **Meteorology** - Weather and atmospheric conditions
3. **Principles of Flight** - Aerodynamics and flight mechanics
4. **Aircraft General** - Aircraft systems and structures
5. **Human Performance & Limitations** - Human factors
6. **Operational Procedures** - SOPs and best practices
7. **Navigation** - Navigation systems and flight planning
8. **Communication** - Radio procedures and phraseology

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Customization

- Update images in `components/` directory
- Modify colors in `tailwind.config.js`
- Edit content in component files
- Update metadata in `app/layout.tsx`
